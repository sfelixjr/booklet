Booklet
=======

By Samuel de Sousa
http://felixjr.org

## About

booklet is a simple but efficient software to generate Booklets.
Booklet has been developed in Python 2.7 with the Qt and Poppler.

## Requirements

* Qt4
* Poppler

## Instructions

```
sudo apt-get install python-qt4 python-poppler-qt4
```
